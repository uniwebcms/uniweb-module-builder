Uniweb Module Builder

Standalone NPM library for building federated modules for Uniweb CMS.

Installation

To install the package, run the following command:

npm install @uniwebcms/module-builder

Usage

Here's an example of how to use your package:

const { getConfigData } = require('@uniwebcms/module-builder');

Documentation

For more information about the package, check out the official documentation on npm.
